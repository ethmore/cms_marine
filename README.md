# Business Website CMS

This project is a Content Management System (CMS) built to manage the dynamic content of the business website. It provides a user-friendly interface for updating services, slider images, business partners, and other essential sections of the site.

## Features

-   Add, edit, and delete website content effortlessly.
-   Manage slider images, about, services and business partners sections.
